# Hashi Kazu Themes

A collection of dark VS Code color themes inspired by Ado's songs.

Adoの楽曲世界をイメージしたVS Code向けダークカラーテーマ集です。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![VS Code](https://img.shields.io/badge/VS%20Code-1.60%2B-blue)

---

## Themes / 収録テーマ

| テーマ名 | イメージ |
|---------|---------|
| **Ado - Hibana** | 「ヒバナ」 |
| **Ado - 心臓** | 「心臓」 |
| **Ado - よだか** | 「よだか」 |

すべて `vs-dark` ベースのダークテーマです。

## Usage / 使い方

1. 拡張機能をインストール
2. `Ctrl+K Ctrl+T`（テーマ選択）を開く
3. 上記のテーマから好きなものを選択
