# ADR-0004: ダークテーマのみの提供

## ステータス
採用

## 確信度
中（全テーマが `"uiTheme": "vs-dark"` を使用していることから推測。コミットメッセージやコメントに明示的な理由の記述はなし）

## コンテキスト
VS Code テーマは `vs-dark`（ダーク）・`vs`（ライト）・`hc-black`（ハイコントラスト）の uiTheme を選択できる。
本拡張機能は歌手 Ado の楽曲イメージをテーマコンセプトとしている。

## 決定
全テーマの `uiTheme` を `vs-dark` とし、ライトバリアントやハイコントラストバリアントは提供しない。

## 理由
Ado の楽曲は夜・深淵・感情の激しさといった暗い情景を喚起するものが多く、ダークテーマがコンセプトと一致する。
（要確認: 技術的制約ではなく美的判断のため、本人の意図を確認することを推奨）

## 捨てた選択肢
- **各テーマのライトバリアントを作成** → 調査時点で痕跡なし。楽曲イメージとの整合性から採用しなかったと推測。
- **ハイコントラスト対応** → 調査時点で痕跡なし。

## 影響範囲
- 新テーマ追加時も `"uiTheme": "vs-dark"` を維持する（明示的にライト対応を決定しない限り）。
